# global-superstore-sales-analysis-powerbi
## Project Title
Global Superstore Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes the Global Superstore dataset to understand sales performance, profit trends, and regional business insights.  
An interactive dashboard is created to help stakeholders track key metrics and identify growth opportunities.

## 🎯 Objective
- Analyze overall sales and profit performance
- Identify top-performing regions and product categories
- Track sales trends across segments and regions
- Provide business insights through interactive visualization

## 🛠 Tools Used
- Microsoft Power BI
- Microsoft Excel

## 📌 Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## 📈 Dashboard Features
- Region-wise sales analysis
- Category and sub-category performance
- Segment-wise sales comparison
- Interactive filters and slicers

## 📂 Dataset
The dataset used in this project is the Global Superstore dataset commonly used for retail sales analysis practice.
